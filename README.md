1  绪论
1.1  研究背景与意义
近年来，随着国家海洋强国战略的持续实施与“21世纪海上丝绸之路”倡议的深入推进，我国海洋经济已进入由传统资源开发向高附加值保障体系转型的快速发展阶段。海洋运输、深海油气钻探、海上风电建设及现代海洋渔业等产业规模的不断扩大，使得海上生产活动的频率与范围显著提升。在这一宏大的战略背景下，海洋环境条件的实时演变对海上作业的安全与效率产生了前所未有的影响。有效波高、风速、风向以及气压等海洋气象要素作为描述海洋环境状态、评估海洋能资源的核心参数，其准确预测直接关系到船舶航行的路径规划、海洋工程结构的疲劳损伤评估以及海上应急搜救的黄金时间窗。据统计，全球多数海上安全事故与极端海洋气象条件下的动力要素演化密切相关 。因此，开展高精度、长时效的多要素海洋气象预测研究，不仅是提升海上综合预警能力、降低海洋灾害风险的技术瓶颈，更是保障国家海洋战略安全、推动海洋经济可持续发展的重大现实需求 。
然而，海洋环境本质上是一个高动态、强随机且高度复杂的非线性动力系统。海洋气象要素的生成与演化并非孤立过程，而是大气环流、表层风场与海洋拓扑结构深度耦合的结果。从物理属性上看，海洋气象序列通常由多个具有显著差异的动力分量复合而成。其中，受季节更替和气候尺度演变影响的部分表现为平稳的长程趋势，具有较强的规律性；受中小尺度天气系统驱动的部分则表现为周期性的季节波演化；而受局地突发性气流或地形扰动影响的部分，则呈现为极强的随机波动与非平稳残差 。这种多尺度特性的交织，使得原始观测序列往往包含大量相互干扰的混杂信息。从空间维度看，海洋气象要素的演化表现出强烈的空间传递性与地理环境依赖性。作为一种连续的动力学系统，海洋气象要素的变化并非孤立发生，而是通过大气环流与海洋波动在空间上进行能量与信息的持续传递 。这种基本的关系表现为，某一空间位置的气象状态往往会对其相邻区域产生直接影响，并随着时间的推移向周边节点扩散 。在理想的开阔海域，这种空间传递性主要遵循大尺度动力过程，能量沿主驱动方向呈辐射状平滑扩散 。然而，在近岸及岛礁密集的复杂海域，这种基本的空间传递路径会受到海岸线形态、岛屿遮蔽效应以及陆地阻隔的剧烈改造 。例如，在我国半岛海域或曲折岸线附近，陆地阻隔效应常导致物理距离相近的两个观测点在空间传递链条上发生物理截断，从而展现出截然不同的动力学演化特征 。这种由空间基本传递性与地理拓扑阻隔性相互交织而成的复杂关联，使得如何有效刻画海洋气象要素在非规则环境下的时空依赖关系，成为了当前预测研究中的关键科学问题 。
回顾海洋气象预测技术的发展历程，主要经历了从经验统计、数值预报到人工智能预报的演进 。早期的经验统计方法虽在平稳序列预测中具有一定参考价值，但在面对海洋要素的强非线性变化时，往往难以获得理想的精度 。基于物理机制的数值预报方法通过离散化处理动力学方程来模拟能量传递，虽具严谨性，但在实际业务中面临计算功耗巨大、实时响应慢以及高度依赖高精度初始场等瓶颈 。
随着深度学习技术的跨越式发展，数据驱动方法为长时序预测提供了新范式，显著提升了对海洋气象序列时序依赖信息的捕捉能力 。近年来，能够处理非规则空间结构的图挖掘技术为海洋建模带来了曙光，其通过显式建模空间拓扑关系，在多个领域展现了卓越性能 。然而，现有的数据驱动模型在应用于海洋这一特殊场景时仍存在明显不足。首先，现有研究普遍缺乏对海洋气象要素多尺度物理特性的深度剥离。 许多模型通常直接对未经处理的原始观测序列进行整体建模，忽略了趋势项、季节项与残差项在时空演化规律上的本质区别 。由于趋势分量往往具有较强的空间相关性，而残差分量则更多受站点局地环境影响，这种“一刀切”的建模方式极易导致不同动力特征的信息混淆，使得模型难以准确捕捉长时演化的确定性规律，也难以应对短时的随机扰动。其次，许多模型在构建空间关系时多采用静态关联方案， 默认关联强度仅取决于物理距离，这忽略了陆地或半岛等地理屏障产生的物理绝缘效应 。在真实海洋环境中，地理阻隔会导致物理距离极近的站点在要素传递上产生截然不同的表现，若不考虑此类物理约束，模型将引入大量的空间关联噪声 。此外，海洋气象系统具有显著的动态特性，站点间的关联随天气模式的改变而实时演进，固定的空间拓扑结构难以充分刻画这种随环境变化的动态传播过程。同时，在处理多要素预测时，往往忽视了动力要素间客观存在的时延关联，未能充分挖掘物理要素间的超前指示信息，导致模型在应对极端突发海况时的预警能力受限 。
综上所述，在复杂地理环境与多变气象条件的双重影响下，如何打破纯数据驱动模型的黑盒局限，通过对复杂序列进行深度解构以分离不同尺度的演化模式，并以此为基础将海洋动力学中的地理约束与动态关联机制有机融入预测架构，已成为当前行业面临的巨大挑战。开发一种能够有效利用物理分量特征、空间拓扑先验知识，并能自适应捕捉要素间复杂时空依赖关系的可靠模型，对于提升海洋气象预测的准确性与稳健性至关重要。这不仅能够为复杂海洋环境下的时空预测问题提供新的建模思路，更对于保障海上作业安全、提升防灾减灾水平具有深远的理论意义与应用价值。
1.2  国内外研究现状
随着海洋气象研究的不断发展，海洋气象预报方法逐渐形成了较为完整的技术体系。总体而言，现有的海洋气象预报方法主要包括经验统计预报方法、数值预报方法以及人工智能预报方法三类。本节将从这三种方法出发，对海洋气象预报领域的国内外研究现状进行系统分析。
